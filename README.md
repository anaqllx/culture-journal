📚 Personal Cultural Journal
Персональний культурний щоденник — це веб-додаток для систематизації та відстеження спожитого культурного контенту (книг, фільмів, ігор, музики). Проєкт розроблений як дипломна робота, що поєднує сучасний стек технологій для зручного менеджменту власних вражень.

🚀 Жива демо-версія: https://culture-journal.vercel.app

✨ Основні можливості
— Мультимедійна бібліотека: Додавання карток із зображеннями, рейтингами та особистими відгуками.
— Кастомні категорії: Можливість створювати власні типи контенту (наприклад, "Подкасти" або "Виставки") з індивідуальними налаштуваннями полів.
— Розумна статистика: Візуалізація активності користувача, розподіл за категоріями та експорт статистики у форматі тексту для поширення.
— Інтуїтивний інтерфейс: Адаптивний дизайн, підтримка світлої та темної тем, зручні модальні вікна для перегляду деталей.
— Безпека: Персоналізований доступ через Google Auth.

🛠 Стек технологій
— Frontend: React + Vite
— Мова: TypeScript
— Стилізація: Tailwind CSS + Shadcn/UI
— Backend / Database: Firebase (Firestore, Auth)
— Зберігання медіа: Cloudinary
— State Management: TanStack Query
— Hosting: Vercel
